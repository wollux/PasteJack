The Problem

Paste doesn't work everywhere.

🖥

IPMI / iLO / iDRAC

Web-based server consoles have no clipboard support. Typing a 64-character SSH key by hand is not an option.

🪟

RDP Sessions

Remote Desktop without clipboard redirection silently blocks paste. You stare at the screen, nothing happens.

📦

VMs without Guest Tools

Fresh VM installs can't share the host clipboard. You need the password to install the tools, but you can't paste the password.

🌐

Paste-Blocked Web Forms

Banking sites, corporate portals, and "security-conscious" forms that intercept paste events with JavaScript.

🔒

Secure Terminals

Compliance environments that disable paste in terminal emulators. Security through inconvenience.

🎯

Any Keyboard Layout

Unicode injection means PasteJack works with US, German, French — any layout. No keymap headaches.

Features

Everything you need. Nothing you don't.

Core

Paste as Keystrokes

Types clipboard contents via CGEvent with configurable speed (5–200ms), adaptive throttling, and a countdown timer so you can focus the target window.

OCR

Copy from Screen

Select any screen region, extract text with Apple Vision — fully on-device. Multi-region support, language detection, editable results.

Core

Type Selected Text

Grab the current text selection and type it into another window. Perfect for transferring text between paste-blocked contexts.

Core

Snippet Library

Save, search, and organize frequently-typed text. Type any snippet with one click — ideal for commands, configs, and credentials you use daily.

Security

Sensitive Content Detection

Warns before typing API keys, private keys, JWT tokens, AWS credentials, or high-entropy strings. Never accidentally leak secrets.

Core

Progress Overlay

Floating HUD shows real-time progress, character count, and Escape-to-cancel hint. macOS notification when typing finishes.

🔐

100% offline. Zero telemetry.

PasteJack never connects to the internet. No analytics, no tracking, no cloud services. Your clipboard data never leaves your Mac. Built with pure Apple frameworks — zero external dependencies.
